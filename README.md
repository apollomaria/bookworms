O BookWorms é um aplicativo que estimula a leitura por meio de competições entre pessoas, utilizando da gamificação como incentivo, através de metas, rankings e comentários.

Surgimos como solução para problemas cada vez mais visíveis em nossa sociedade, como o analfabetismo funcional e queda no número de leitores.

LINK: https://apollomaria.github.io/bookworms/
